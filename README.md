## কখন নথি তলব হয়?

সাধারণত নিচের পরিস্থিতিতে হয়ঃ

১। আপিল হলে

নিচের আদালতের রায়ের বিরুদ্ধে আপিল করলে উপরের আদালত মূল নথি চায়।
উদাহরণঃ সহকারী জজ আদালতের রায়ের বিরুদ্ধে জেলা জজ আদালতে আপিল করা হয়েছে।
তখন জেলা জজ আদালত নিচের আদালতের নথি তলব করতে পারে।

২। রিভিশন হলে

তখন নথি তলব হয়।
